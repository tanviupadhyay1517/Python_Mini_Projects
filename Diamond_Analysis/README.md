# 💎 Diamond Analysis

## 📌 Overview
Exploratory Data Analysis (EDA) on a diamond dataset to identify the relationship between diamond features and price using Python.

## 🛠️ Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## 📊 Analysis Includes
- Data Cleaning
- Price Distribution
- Carat vs Price Analysis
- Cut-wise Distribution
- Data Visualization

## 📂 Files
- `diamond_analysis.ipynb`
- `diamonds.csv`
- `Histogram Chart.png`
- `Scatter Chart.png`
- `Pie Chart.png`

## 🚀 Skills Demonstrated
EDA • Data Visualization • Statistical Analysis • Python Programming
