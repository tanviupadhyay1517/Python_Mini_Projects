# Instagram Reach Analysis

This project analyzes Instagram post performance using Python and Machine Learning.

## Features

- Data Cleaning and Preprocessing
- Exploratory Data Analysis (EDA)
- Reach Source Analysis
- Correlation Analysis
- Caption Word Cloud
- Engagement Analysis
- Follow Conversion Rate Analysis
- Machine Learning Prediction Model

##  Libraries Used

- Pandas
- NumPy
- Matplotlib
- Seaborn
- Plotly
- WordCloud
- Scikit-Learn

## 📊 Key Insights

- Home feed generates the highest impressions.
- Hashtags contribute significantly to reach.
- Likes have a strong relationship with impressions.
- Engagement metrics influence post performance.
- Follow conversion rate measures audience growth effectiveness.

## 🤖 Machine Learning

Model Used:
- PassiveAggressiveRegressor

Target Variable:
- Impressions

## 📈 Results

The model predicts Instagram post impressions using engagement metrics such as:
- Likes
- Comments
- Shares
- Saves
- Profile Visits

Instagram Reach Dataset
